# Kurzbeschreibung des Projekts

Im Modul Automatisierungstechnik besteht die Aufgabe darin, in einer Gruppe innerhalb eines vorgegebenen Zeitraums ein technisches System zu entwickeln und umzusetzen. Zentrum des Projekts ist die Konzeption, Konstruktion und Programmierung eines CNC-Hotwire-Foam-Cutters, also einer Maschine zum automatisierten Schneiden von Schaumstoff mithilfe eines erhitzten Drahts.



Neben der Entwicklung eines funktionsfähigen Prototyps gehört auch eine umfangreiche technische Dokumentation zum Projekt. Darin sollen das Systemkonzept, die technischen Entscheidungen sowie die Umsetzung nachvollziehbar beschrieben werden. Ziel ist es, zentrale Inhalte der Automatisierungs- und Steuerungstechnik praktisch anzuwenden und den vollständigen Entwicklungsprozess eines automatisierten Systems nachzuvollziehen.
