\chapter{Bedienung}

Nach der Inbetriebnahme kann ELMAR für den Schneidvorgang verwendet werden.

\begin{enumerate}
	
	\item Befestigen Sie den Rohling auf der vorgesehenen Dornaufnahme im Arbeitsbereich von ELMAR. 
	Achten Sie darauf, dass der Rohling fest sitzt und an den Markierungen ausgerichtet ist.
	
	\item Wählen Sie eine geeignete Schneiddatei im G-Code-Format auf dem Computer aus. 
	
	
	\item Laden Sie die Schneiddatei über die Steuerungssoftware auf ELMAR. 
	Wählen Sie dazu die entsprechende Datei in der Software aus und starten Sie den Upload.
	
	\input{../../tikz/Eigene_Bilder/Inbetriebnahme/Datei_hochladen}
	
	\item Warten Sie, bis der erfolgreiche Upload auf der Anzeige bestätigt wird.
	
	\input{../../tikz/Eigene_Bilder/Inbetriebnahme/Datei_erkannt}
	
	\item Drücken Sie den Start-Knopf, um den Schneidvorgang zu beginnen.
	
	\input{../../tikz/Eigene_Bilder/Inbetriebnahme/Prozess_gestartet}
	
	\item Die Anzeige zeigt den Fortschritt des Schneidvorgangs an.
	
	\input{../../tikz/Eigene_Bilder/Inbetriebnahme/Prozess_im_Gange}
	
	\item Mit dem Pause-Knopf kann der Schneidvorgang jederzeit pausiert werden.
	
	\input{../../tikz/Eigene_Bilder/Inbetriebnahme/Prozess_pausiert}
	
	\item Mit dem Stopp-Knopf wird der Schneidvorgang beendet und ELMAR fährt in die Ausgangsposition (Siehe Homing).
	
	\input{../../tikz/Eigene_Bilder/Inbetriebnahme/Prozess_gestoppt}
	
	\item Nach Abschluss zeigt die Anzeige das Ende des Prozesses an.
	
	\input{../../tikz/Eigene_Bilder/Inbetriebnahme/Prozess_abgeschlossen}
	
	\item Warten Sie mindestens 10 Sekunden, bis der Draht abgekühlt ist, bevor Sie das Werkstück entnehmen. Die Anzeige gibt an, wie viel Wartezeit noch erforderlich ist.
	
	\input{../../tikz/Eigene_Bilder/Inbetriebnahme/Draht}
	
\end{enumerate}
